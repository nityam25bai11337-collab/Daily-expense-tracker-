# Daily Expense Tracker (Python Project)

This is a simple Python project made for my First Semester.  
The purpose of this program is to store daily expenses and calculate the total money spent.

 Objective
To keep track of how much money we spend every day in different categories like food, travel, shopping etc.

Tools and Technology used
- Python (Basic)
- Functions
- List
- While loop
- Input/Output

 Features
- Add new expense with amount and category
- View all added expenses
- Calculate total amount spent
- Simple menu-driven program
